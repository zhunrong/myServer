# mysql常用指令

>注意：所有指令应该以";"结尾

>在user表中添加用户后需要刷新权限(FLUSH PRIVILEGES)

1. 查看数据库
```
show databases;
```
2. 创建数据库
```
create database 数据库名;
```
3. 删除数据库
```
drop database 数据库名;
```
4. 选择数据库
```
use 数据库名字;
```
5. 查看数据表
```
show tables;
```
6. 显示表的结构
```
describe 表的名字;
```
7. 删除表
```
drop table 表的名字;
```
